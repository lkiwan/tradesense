# TradeSense AI - Changelog

This file documents all significant changes made to the TradeSense AI trading platform.

## Dashboard Upgrade
**Date: December 19, 2024**

Major upgrade to the Dashboard page with new stats, charts, and improved visual design:

### New Components
- **StatCard** - Animated stat cards with gradient icons and hover effects
- **MiniPerformanceChart** - SVG line chart showing cumulative profit over trades

### Stats Cards (Header)
- **Profit Total** - Shows total PnL with percentage trend indicator
- **Win Rate** - Win percentage with wins/total ratio
- **Best Trade** - Biggest winning trade amount (also shows worst trade)
- **Positions** - Open positions count + closed trades count

### Sidebar Additions
- **Performance Chart** - Mini chart showing profit curve over time
- **Quick Trade Buttons** - Fast symbol selection (AAPL, TSLA, BTC-USD, etc.)

### Visual Improvements
- Gradient icons on all stat cards
- Shadow and hover effects on cards
- Better chart section with rounded borders
- Enhanced open positions list with improved PnL display
- Cleaner trade history table with hover states
- Plan type badge (Starter/Pro/Elite) with color coding
- Improved button styling throughout

## Landing Page Upgrade
**Date: December 2024**

Completely redesigned the landing page to create a modern, professional, and engaging first impression. The new landing page includes:

- **Animated Hero Section**: Added typing text animation cycling through "Professionnel", "Finance", "Rentable", "Expert" with a pulsing AI badge and animated background gradients
- **Live Price Ticker**: Marquee-style banner showing real-time prices for BTC, ETH, AAPL, TSLA, IAM, NVDA, GOOGL, and ATW
- **Animated Statistics**: Counter animations that trigger on scroll showing 10,000+ traders, $2M+ distributed, 85% success rate, and 80% profit split
- **Features Section**: Four feature cards (AI Signals, News, Community, Education) with hover effects and scroll animations
- **How It Works**: Step-by-step process with 4 stages showing the journey from choosing a plan to getting funded
- **Testimonials**: Three trader testimonials including multilingual support (Arabic and English) with star ratings
- **Benefits Section**: Interactive dashboard preview mockup with gradient background
- **FAQ Accordion**: Five frequently asked questions about prop trading with expandable answers
- **Final CTA Section**: Limited offer banner with call-to-action buttons


## Bug Fixes
**Date: December 22, 2024**

- **Ad-Blocker Fix**: Renamed "Affiliate" page to "Partners" to prevent browser ad-blockers from blocking the application resource (net::ERR_BLOCKED_BY_CLIENT), which was causing a blank page on load.
